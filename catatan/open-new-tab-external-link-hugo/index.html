Open New Tab External Link Hugo

12 November 2020


Catatan Ara kali ini akan berbagi sedikit mengenai hugo, bagaimana caranya membuat external link dibuka pada tab baru, langsung saja tanpa basa basi, buat file render-link.html pada folder layouts theme teman teman layouts/_default/_markup/render-link.html, untuk isi render-link.htmlnya sendiri kurang lebih seperti ini :

<a href="{{ .Destination | safeURL }}"{{ with .Title}} title="{{ . }}"{{ end }}{{ if strings.HasPrefix .Destination "http" }} rel="noopener noreferrer" target="_blank"{{ end }}>{{ .Text }}</a>
