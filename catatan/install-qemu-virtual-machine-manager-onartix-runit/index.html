Install Qemu Virtual Machine Manager On Artix Runit

09 June 2021


Pada catatan ini kita akan menginstall qemu dan virtual machine manager pada sistem operasi Artix Linux (runit) langsung saja seperti biasa tanpa panjang lebar. Pertama pastikan repo [extra], [community], dan [multilib] dari Arch Linux telah di aktifkan. Langsung saja install paket-paket yang dibutuhkan seperti berikut :

 # pacman -S qemu virt-manager libvirt-runit vde2 dnsmasq ebtables bridge-utils openbsd-netcat

konfigurasi pada /etc/libvirt/libvirtd.conf uncomment pada baris berikut :

 unix_sock_group = "libvirt"
 unix_sock_rw_perms = "0770"

tambahkan group libvirt

 # usermod -a -G libvirt $(whoami)
 newgrp libvirt

aktifkan service libvirtd, virtlockd , virtlogd

 # ln -s /etc/runit/sv/libvirtd /etc/runit/runsvdir/default
 # ln -s /etc/runit/sv/virtlockd /etc/runit/runsvdir/default
 # ln -s /etc/runit/sv/virtlogd /etc/runit/runsvdir/default
