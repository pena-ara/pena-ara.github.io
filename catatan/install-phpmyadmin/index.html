Install Phpmyadmin

19 October 2020


Catatan-catatan sebelumnya kita sudah menginstall Apache, MySQL dan juga kita sudah menginstall php, pada catatan ini kita akan mencoba menginstall phpMyadmin, phpMyadmin digunakan untuk mempermudah penggunaan mysql, langsung saja seperti biasa buka terminal kesayangan kita semua dan masukan perintah sudo pacman -S phpmyadmin, tunggu sampai proses instalasi selesai, sesudah installasi selesai lakukan beberapa konfigurasi pada php kita, aktifkan extensi bz2 dan mysqli dengan cara menghapus tanda ; pada file /etct/php/php.ini kurang lebih akan menjadi seperti ini :

extension=bz2.so
extension=mysqli.so

Selanjutnya buat konfigurasi baru untuk phpMyadmin kita dengan cara buat file konfigurasi dengan nama phpmyadmin.conf dan letakan di /etc/httpd/conf/extra/, kurang lebih seperti ini isi konfigurasinya :

Alias /phpmyadmin "/usr/share/webapps/phpMyAdmin"
<Directory "/usr/share/webapps/phpMyAdmin">
DirectoryIndex index.php
AllowOverride All
Options FollowSymlinks
Require all granted
</Directory>

Belum selesai :), tahapan terakhir tambahkan konfigurasi phpmyadmin kita yang sudah dibuat tadi kedalam konfigurasi apache, letak konfigurasi apache ada di /etc/httpd/conf/httpd.conf, letakan konfigurasi phpmyadmin diakhir baris konfigurasi apache kita kurang lebih seperti ini :

Include conf/extra/phpmyadmin.conf

Jika semua sudah siap restart apache kita dengan perintah systemctl restart httpd. Selamat phpMyadmin sudah terinstall dan dapat diakses melalui localhost/phpmyadmin. :)
