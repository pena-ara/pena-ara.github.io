Mengelola File Pada Gnu Linux

09 October 2020


Pada catatan sebelumnya kita sudah sedikit belajar mengenai 10 perintah dasar untuk mengelola direktori di sistem opersai gnu/linux, pada kesempatan ini kita akan sedikit belajar perintah dasar mengelola file di sistem opersai gnu/linux, ok langsung saja tanpa panjang lebar mari kita belajar bersama apa saja perintah dasar untuk mengelola file di sistem opersai gnu/linux.

chgrp Perintah ini sudah dijelaskan pada catatan sebelumnya.mengelola directori

Perintah cp juga sudah dijelaskan pada kesempatan sebelumnya, namaun pada kesempatan ini akan sedikit di jabarkan, ada beberapa penggunaan perintah cp untuk mengelola file di gnu/linux anatar lain:

cp -a => digunakan untuk menyimpan atribut-atribut file

cp -b => digunakan untuk membackup file

cp -d => digunakan untuk membuat link file

cp -i => digunakan untuk menampilkan pesan sebelum file dicopy

cp -R => digunakan untuk mengcopy semua file beserta direktori secara recursive

cp -v => digunakan untuk menampilkan pesan saat pengcopyan berlangsung

ln Perintah ini digunakan untuk membuat nama baru pada suatu file.

mv Perintah ini digunakan untuk memindahkan suatu file

Perintah rm digunakan untuk menghapus suatu file, adapun penggunaan perintah rm antara lain:

rm* => digunakan untuk menghapus semua file yang ada didalam direktori

rm -f => digunakan untuk menghapus file tanpa pesan

rm -R => digunakan untuk menghapus file-file dalam semua subdirektori dan direktori utamanya

Mungkin itu saja yang bisa kita pelajari sedikit mengenai perintah dasar untuk mengeleloa file di sistem opersai gnu/linux apa bila kawan-kawan ada tambahan silahkan di kolom komentar. Semoga bisa bermanfaat bagi kita semua.
