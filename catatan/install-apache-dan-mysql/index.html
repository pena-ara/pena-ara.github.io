Install Web Server Apache Dan DBMS Mysql

10 October 2020


Sebelum ke proses installasinya ara akan menjelaskan sedikit tentang web server, apa itu web server ? web server adalah sebuah software yang meberikan layanan berbasis data dan berfungsi menerima permintaan dari HTTP atau HTTPS pada client yang biasa kita sebut web browser dan untuk mengirimkan kembali dalam bentuk halaman web dan pada umumnya akan berbentuk HTML.

sekarang sudah tahukan apa itu web server ?, yang akan kita install sekarang adalah salah satu contoh web server yang lumayan banyak penggunanya yakni apache, langsung saja keproses installasinya, disini ara menggunakan arch linux untuk menginstall apache.

Pertama buka terminal dan ketikan perintah sudo pacman -S apace untuk menginstall apache.

Setelah proses install berhasil aktifkan service apache dengan perintah sudo systemctl enable httpd dan untuk menjalankannya bisa menggunakan perintah sudo systemctl start httpd.

sampai disini apache sudah terinstall dan berjalan di sistem operasi kita, untuk mengetesnya buatlah file index.html di folder /srv/http/.

<html>
<head>
	<tittle>Welcome</title>
</head>
<body>
	<h2>Welcome</h2>
</body>
</html>

kemudian buka web browser kesayangan dan masukan alamat localhost atau 12.0.0.1 jika berhasil maka akan tampil tulisan Welcome.

tidak hanya menginstall web server apache disini juga kita akan menginstall MySQL. apa itu MySQL ? MySQL adalah sebuah software database management sysytem (DBMS). Langsung saja ke proses penginstallannya :

Seperti biasa buka terminal kesayangan kita dan ketikan perintah sudo pacman -S mysql.

Setelah proses installasi berhasil jangan lupa aktifkan servicenya dengan cara sudo systemctl enable mysqld dan jalankan dengan cara sudo systemctl start mysqld, apabila terjadi error saat proses mengaktikannya bisa menngunakan perintah berikut sudo mysql_install_db --user=mysql --basedir=/usr --datadir=/var/lib/mysql.

Untuk memberikan password pada mysql bisa menggunakan perintah sudo mysql_secure_installation.

mudah bukan install Web Server Apache dan DBMS MySQL di sistem operasi Arch Linux.
