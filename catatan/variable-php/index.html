Pada catatan sebelumnya kita sudah mengerti sedikit tentang komentar php, dicatatan ini kita akan belajar sedikit mengenai variable di php, sebelumnya apa itu variable ?, variable merupakan memori sementara untuk menyimpan nilai suatu data. Penulisan variable di php diawali dengan tanda $ dan dilanjutkan dengan nama variablenya, ada beberapa aturan penulisan variable di php, antara lain sebagai berikut :

  • Nama variable harus diawali dengan huruf atau _.
  • Nama variable hanya bisa menggunakan karakter a-z, A-Z, 0-9, dan _.
  • Nama variable tidak boleh mengandung spasi.
  • Penamaan variable bersifat case censitive.

Didalam php variable dibagi menjadi tiga yaitu, variable string, variable numerik, dan variable array.

Variable String
Variable string digunakan untuk menyimpan nilai data teks yang diawali dengan ' 'atau " ", contohnya kurang lebih seperti ini :

<?php
  $nama = "Ara";
  echo "Nama saya $nama";
?>

Variable Numerik
Variable numerik digunakan untuk menyimpan nilai data numerik atau angka, langsung saja kecontohnya :

<?php
  $angka = 1;
  $angka2 = 3;
  $jumlah = $angka * $angka2;
  echo $jumlah;
?>

Variable Array
Variable array untuk menyimpan nilai data array, apa itu array ?, array merupakan tipe data tersetruktur yang berguna untuk menyimpan sejumlah data yang bertipe sama. Bagian yang menyusun array dinamakan elemen array, dari tiap-tiap elemen array tersebut dapat diakses tersendiri melalui indexs array, index array dimulai dari 0. Contoh penulisan variable array kurang lebih seperti ini:

<?php
  $buah = array('pepaya','mangga','pisang');
  echo $buah[1];
?>

maka yang akan tampil adalah mangga, kenapa mangga ?, karena nilai dari indexs 1 adalah mangga. :)
