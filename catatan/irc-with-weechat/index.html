Irc With Weechat

30 December 2020


Dalam catatan ini kita akan mencoba irc dengan menggunakan weechat, dicatatan ini kita akan mengkonfigurasikan autoconnect to freenode, dan konfigurasi SASL ECDSA-NIST256P-CHALLENGE, langsung saja tanpa panjang lebar install terlebih dahulu Weechat menggunakan sudo pacman -S weechat, jikas sudah terinstall langsung saja pertama kita akan mengkonfigurasi autoconnect to freenode, masuk ke weechat dengan mengetikan weechat di terminal, untuk mengkonfigurasinya cukup masukan baris-baris perintah berikut ini didalam weechat :

/server add freenode chat.freenode.net/6697 -ssl
/set irc.server.freenode.addresses "chat.freenode.net/6697"
/set irc.server.freenode.ssl on
/set irc.server.freenode.username <username>
/set irc.server.freenode.autoconnect on
/save

sekarang coba keluar dan masuk kembali ke dalam weechat, dan apa yang terjadi maka ketika kita masuk ke weechat, kita otomatis sudah berada di freenode :), sekarang kita akan mengkonfigurasi SASL ECDSA-NIST256P-CHALLENGE, tujuan mengkonfigurasi SASL ini adalah untuk login otomatis tanpa memasukan password irc kita, dalam catatan ini ara menggunakan akun freenode, bagi yang belum punya silahkan daftar terlebih dahulu Nickname Registration, jikas sudah kembali ke weechat, sebelum kembali ke weechat kita buat public key terlebih dahulu untuk ECDSA, langsung saja ke pembuatan keynya kembali keterminal dan masukan baris perintah berikut :

openssl ecparam -genkey -name prime256v1 -out ~/.weechat/ecdsa.pem
dan untuk melihat public keynya kita gunakan perintah berikut,
openssl ec -noout -text -conv_form compressed -in ~/.weechat/ecdsa.pem | grep '^pub:' -A 3 | tail -n 3 | tr -d ' \n:' | xxd -r -p | base64
copy public key yang muncul.

jika sudah membuat public key, saatnya kembali lagi ke weechat, dan kita akan memulai konfigurasi sasl nya.

/nick <nick yang sudah di daftarkan>
/msg nickserv identify <password>
/msg nickserv set pubkey <isi public key yang sudah dicopy tadi>
/set irc.server.freenode.sasl_mechanism ecdsa-nist256p-challenge
/set irc.server.freenode.sasl_username <username>
/set irc.server.freenode.sasl_key "%h/ecdsa.pem"
/save
/reconnect freenode

Selamat bergabung di IRC kembali…… :), berikut irc #archlinux di server freenode
