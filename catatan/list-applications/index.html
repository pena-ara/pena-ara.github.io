List Applications

25 May 2021


List aplikasi yang di install setelah menginstall distro gnu/linux (core) :

System Tools

  • alacritty => Terminal emulator
  • pcmanfm => File manager
  • ranger => File manager
  • dwm => Window manager

Preferences

  • lxappearance => Customizes look & feel
  • qt5ct & qt5-styleplugins => Customizes look & feel

Sound & Video

  • mpv => Media player
  • moc => Audio player
  • simplescreenrecorder => Screen recorder
  • cinelerra => Video editing
  • kdenlive => Video editing

Internet

  • qutebrowser => Browser
  • mutt => E-mail client
  • aria2 => Download manager
  • speedtest-cli => Testing internet bandwidth
  • telegram-desktop => Cloud based messaging
  • zoom => Video conferencing

Graphics

  • inkscape => Vektor editor
  • gimp => Raster editor

Office

  • libreoffice => Office

Games

  • steam => Games store

Virtual

  • qemu & virt-manager => Virtual Machine Manager

Accessories

  • neovim => Text editor
  • nitrogen => Background browser
  • xarchiver => Archivers
  • xpdf => PDF viewer
  • sxiv => Image viewer
  • flameshot => Screenshot
