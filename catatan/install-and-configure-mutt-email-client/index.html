Install and Configure Mutt E-mail Client

13 June 2021


Pada catatan ini kita akan menginstall dan mengkonigurasi mutt dengan akun gmail, langsung seperti biasa install terlebih dahulu muttnya dengan cara

 # pacman -S mutt

buat beberapa folder dan file untuk konfigurasi :

 mkdir -p ~/.mutt/cache/headers
 mkdir ~/.mutt/cache/bodies
 touch ~/.mutt/certificates
 touch ~/.mutt/muttrc

isi file ~/.mutt/muttrc

 # IMAP
 set imap_user = <username>@gmail.com
 set imap_pass = <password>
 set spoolfile = imaps://imap.gmail.com/INBOX
 set folder = imaps://imap.gmail.com/
 set record="imaps://imap.gmail.com/[Gmail]/Sent Mail"
 set postponed="imaps://imap.gmail.com/[Gmail]/Drafts"
 set mbox="imaps://imap.gmail.com/[Gmail]/All Mail"
 set header_cache = "~/.mutt/cache/headers"
 set message_cachedir = "~/.mutt/cache/bodies"
 set certificate_file = "~/.mutt/certificates"
 # SMTP
 set smtp_url = "smtp://<username>@smtp.gmail.com:587/"
 set smtp_pass = $imap_pass
 set ssl_force_tls = yes # Require encrypted connection
 # Editing
 set editor = "nvim"
 set edit_headers = yes
 set charset = UTF-8
 # Sender
 unset use_domain
 set realname = "<name>"
 set from = "<username>@gmail.com"
 set use_from = yes

berikan akses mutt Less secure app access

 Allow less secure apps: ON
