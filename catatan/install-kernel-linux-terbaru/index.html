Install Kernel Linux Terbaru

14 October 2020


Catatan kali ini ara akan membagikan bagaimana cara menginstall kernel linux versi terbaru di sistem operasi arch linux, saat catatan ini di buat kernel linux terbaru memasuki versi 5.9. Kita bisa cek versi dari kernel linux ini di situs resminya kernel.

Langsung saja ketopik utama, oh iya sekedar menginfokan pada saat catatan ini di buat KDE Plasma baru saja merilis versi barunya yakni 5.20, jadi langsung saja kita install keduanya :).

Sebelum kita menginstall keduanya, eh sebenarnya sih bukan menginstall ya lebih tepatnya mengupdate atau mengupgrade karena saat ini ara menggunakan kernel linux versi 5.8 dan KDE Plasma versi 5.19, untuk mengupgrade keduanya kita membutuhkan atau mengaktifkan repositories testing pada arch linux kita dengan cara mengedit konfigurasi pacman, konfigurasi pacman dapat di temukan di /etc/pacman.conf kemudian hilangkan tanda pagar pada repositories testing, kurang lebih jadi seperti ini:

[testing]
Include = /etc/pacman.d/mirrorlist

Setelah mengaktifkan repositories testing kita bisa mengupgrade kernel, aplikasi, dan paket ke versi terbaru dengan cara memasukan perintah sudo pacman -Syyu di terminal kesayangan kita. Perlu di ketahui ketika kita menggunakan kernel, aplikasi, dan paket versi terbaru seringkali kita akan mendapat masalah atau error pada sistem operasi kita jadi siapkan terlebih dahulu jiwa petualang kita :).
